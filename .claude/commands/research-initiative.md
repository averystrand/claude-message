Perform initiative research for $ARGUMENTS.

Include:

- Initiative Description
- Key Compelling Events
- Impact on Priorities and Budgets
- Impact on Teams and Individuals
- Expected Capabilities
- Measurement KPIs
- Relevant Industry Trends

When complete, write a summary document to a markdown file in the existing `/research/initiatives/` directory
